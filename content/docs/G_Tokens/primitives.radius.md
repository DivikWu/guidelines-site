# 🔲 Radius Primitives

> [!TIP]
> 圆角系统塑造产品的视觉风格与品牌调性,统一的圆角规范能提升界面的精致感。

---

## 圆角定义

| Token | 数值 | 应用场景 |
|-------|------|----------|
| `radius-xs` | 2px | 小型标签、徽章 |
| `radius-sm` | 4px | 输入框、小按钮 |
| `radius-md` | 8px | 卡片、对话框 |
| `radius-lg` | 12px | 大型卡片、面板 |
| `radius-xl` | 16px | 特殊强调元素 |
| `radius-full` | 9999px | 圆形头像、药丸按钮 |
