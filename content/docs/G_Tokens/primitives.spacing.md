# 📏 Spacing Primitives

> [!TIP]
> 间距系统基于 **8px 基准单位**,确保界面元素的一致性与节奏感。

---

## 间距定义

| Token | 数值 | 用途示例 |
|-------|------|----------|
| `spacing-050` | 4px | 紧凑元素间距 |
| `spacing-100` | 8px | 小间距、内边距 |
| `spacing-150` | 12px | 中小间距 |
| `spacing-200` | 16px | 标准间距、卡片内边距 |
| `spacing-300` | 24px | 大间距、区块间距 |
| `spacing-400` | 32px | 超大间距、页面分段 |
| `spacing-500` | 40px | 页面级间距 |
| `spacing-600` | 48px | 大型区块分隔 |
