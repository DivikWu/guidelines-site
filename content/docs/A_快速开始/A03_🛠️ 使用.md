# 🔫 Figma 使用

## 在 Figma 中快速开始

- 打开 [YAMI UI/UX Guidelines](https://www.figma.com/design/6oOAy72DBff4P6NzJYc2hi/YAMI-UI-UX-Guidelines?node-id=2290-1175&t=SsnNzInOnhyAx5F6-1) Figma 文件
- 启用所需的组件库与样式库
- 从组件和样式开始设计，而不是从空白画布开始

> 设计系统是设计的起点，而不是设计完成后的对照物。

## 使用组件

- 从官方组件库中插入组件
- 非必要不 Detach，保持与系统同步
- 使用 Variant 表达不同状态与尺寸
- 方案确认并进入开发前，优先回归标准组件

## 使用 Variant 与状态

- 使用 Variant 表达默认、悬浮、激活、禁用等状态
- 不要通过手动修改样式来"模拟状态"
- 相同类型组件应保持一致的状态逻辑

> 如果需要新增状态，请推动系统演进，而不是临时绕过。

## 使用样式与 Token

- 始终使用系统提供的颜色、文字和效果样式
- 避免直接填写颜色值或字号
- 基于 Token 的样式可以保证一致性并降低维护成本

## 更新与同步

- 定期检查组件库更新
- 在明确原因前，建议同步更新
- 大范围使用前先了解变更内容
