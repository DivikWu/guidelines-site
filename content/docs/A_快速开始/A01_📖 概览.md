# 📖 概览

用于定义设计系统的整体结构、核心原则与使用边界，作为所有设计与研发协作的统一入口。


## 设计系统包含哪些内容？

- 🎯 指导设计决策的核心设计原则
- 🌟 定义品牌一致性的视觉识别系统
- 🎨 颜色、字体、间距、布局等基础规范
- 🧩 可复用的组件体系与使用规则
- 📝 指导文案与信息架构的内容策略
- 🛠️ 包含 Token 与工具的设计资源
- 📐 常见业务场景的设计模式与最佳实践


## 这套设计系统适用于：

- 产品设计师（UX / UI）
- 前端与客户端工程师
- 产品经理及相关协作角色
