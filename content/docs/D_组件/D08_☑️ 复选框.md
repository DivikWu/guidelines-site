# ☑️ 复选框

复选框（Checkbox）用于多选场景，用户可选择一项或多项，选项之间互不排斥。

## 组件概述

复选框提供明确的多选交互，选中态清晰可见。支持单独使用或成组使用，适用于列表多选、协议勾选、设置项等场景。

## 组件类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 单个复选框 | 独立使用，控制单一开关 | 协议同意、单选项开关 |
| 复选框组 | 多个选项成组，可多选 | 多选列表、标签筛选 |

## 状态

| 状态 | 描述 |
|------|------|
| 未选中 | 默认状态，可点击 |
| 选中 | 已选择，显示勾选图标 |
| 半选中 | 组内部分选中时，父级展示半选态 |
| 禁用 | 不可点击，视觉弱化 |

## 使用规则

**何时使用**
- 用户需要选择多项
- 选项可多选且互不排斥
- 协议勾选、批量操作、筛选条件等

**何时不使用**
- 只能选一项时，使用 [[D09_⚪ 单选框|单选框]]
- 二态开关时，使用 [[D10_🔘 开关|开关]]
