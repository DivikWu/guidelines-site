# 🔘 开关

开关（Switch）用于二态切换，用户在「开」与「关」之间切换，切换后立即生效。

## 组件概述

开关通过滑动或点击实现状态切换，视觉反馈明确，适用于功能开关、设置项、权限控制等即时生效的场景。

## 组件类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 标准开关 | 滑块 + 轨道，左右切换 | 通用二态设置 |
| 带文字开关 | 开关旁或上方附带说明文字 | 需要解释含义的设置 |
| 小尺寸开关 | 紧凑尺寸 | 表格内、列表项、移动端 |

## 状态

| 状态 | 描述 |
|------|------|
| 关闭 | 默认关闭态，滑块在左侧 |
| 开启 | 开启态，滑块在右侧，轨道高亮 |
| 禁用 | 不可点击，视觉弱化 |
| 加载中 | 切换中，可展示加载态防止重复点击 |

## 使用规则

**何时使用**
- 二态开关，如开启/关闭、是/否
- 设置项、权限、通知等即时生效的场景
- 移动端或需快速切换的场景

**何时不使用**
