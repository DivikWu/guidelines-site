# ⚪ 单选框

单选框（Radio）用于单选场景，用户只能在一组选项中选择一个，选项之间互斥。

## 组件概述

单选框通过圆形选择器与清晰选中态，帮助用户在互斥选项中做出唯一选择，适用于支付方式、配送方式、地址选择等场景。

## 组件类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 标准单选框 | 圆形选择器 + 文案，横向排列 | 选项较少、列表式展示 |
| 卡片式单选 | 选项以卡片形式展示，选中时有边框或背景 | 需突出选项内容、对比展示 |

## 状态

| 状态 | 描述 |
|------|------|
| 未选中 | 默认状态，可点击 |
| 选中 | 已选择，中心圆点高亮 |
| 禁用 | 不可点击，视觉弱化 |

## 使用规则

**何时使用**
- 只能选择一个选项
- 支付方式、配送方式、地址、类型选择等
- 选项互斥且需要明确对比时

**何时不使用**
- 可多选时，使用 [[D08_☑️ 复选框|复选框]]
- 选项较多或需搜索时，考虑 [[D11_📋 下拉菜单|下拉菜单]]
