# 📋 下拉菜单

下拉菜单（Dropdown/Select）通过点击触发展开选项列表，在有限空间内提供大量选项的选择能力。

## 组件概述

下拉菜单收起时占用空间小，展开后展示完整选项，适用于表单选择、筛选、地区选择等场景。支持单选、多选、搜索、分组、级联等能力。

## 组件类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 单选下拉 | 从选项中选择一项 | 地区、类型、状态选择 |
| 多选下拉 | 可选择多项，已选展示标签 | 标签、分类多选 |
| 搜索下拉 | 支持关键词搜索过滤选项 | 选项较多、需快速定位 |
| 级联下拉 | 多级联动选择 | 省市区、多级分类 |

## 状态

| 状态 | 描述 |
|------|------|
| 默认 | 未展开，展示占位符或已选值 |
| 展开 | 选项列表已展开 |
| 禁用 | 不可点击，视觉弱化 |

## 设计要点

- **选项分组**：选项可分组展示，组标题与选项样式区分
- **搜索过滤**：选项多时提供搜索，实时过滤
- **虚拟滚动**：选项极多时使用虚拟滚动保证性能
- **选中反馈**：已选项需明确标识

## 使用规则

**何时使用**
- 选项较多（超过 5 个）或空间有限
- 需要从预设列表中单选或多选
- 表单选择、筛选器、地区选择等

**何时不使用**
- 选项少于 5 个且空间充足时，可考虑 [[D09_⚪ 单选框|单选框]] 或 [[D08_☑️ 复选框|复选框]]
- 二态切换使用 [[D10_🔘 开关|开关]]
