# 📝 输入框

输入框（Input）是用户输入文本的基础组件，用于收集姓名、邮箱、搜索词等信息。

## 组件概述

输入框承载文本输入能力，需保证标签清晰、占位符友好、错误提示及时，并与 [[C02_🔤 字体排版|字体]]、[[C03_📏 间距|间距]] 等基础规范一致。

## 组件类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 单行输入 | 单行文本，可带前后缀 | 姓名、手机号、短文本 |
| 多行输入 | 多行文本，可调整高度 | 备注、描述、长文本 |
| 密码输入 | 内容以圆点或星号隐藏，可切换显示 | 密码、敏感信息 |
| 搜索输入 | 带搜索图标，支持清除 | 搜索框、筛选关键词 |

## 状态

| 状态 | 描述 |
|------|------|
| 默认 | 未聚焦、可输入 |
| 聚焦 | 获得焦点，边框或背景高亮 |
| 禁用 | 不可编辑，视觉弱化 |
| 错误 | 输入有误，显示错误提示和错误态样式 |
| 成功 | 校验通过，可选展示成功态 |

## 设计要点

- **标签**：明确说明输入内容，必填项可加星号
- **占位符**：提供示例或提示，避免作为唯一说明
- **帮助文本**：补充说明、字数限制等
- **错误提示**：靠近输入框，说明具体错误原因

## 使用规则

**何时使用**
- 需要用户输入文本、数字等
- 搜索、登录、表单填写等场景

**何时不使用**
- 选项固定且较少时，使用 [[D11_📋 下拉菜单|下拉菜单]] 或 [[D09_⚪ 单选框|单选框]]
- 二态开关时，使用 [[D10_🔘 开关|开关]]
