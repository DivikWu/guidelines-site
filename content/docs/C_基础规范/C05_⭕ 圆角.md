# ⭕ 圆角

圆角赋予界面元素柔和与现代感，是品牌视觉语言的重要组成部分。YAMI 设计系统定义了统一的圆角 Token，确保各组件之间的视觉协调。

---

## 圆角 Token

设计系统提供以下圆角层级：

| Token | 数值 | 用途说明 |
|-------|------|----------|
| radius/none | 0px | 直角，无圆角 |
| radius/small | 4px | 小圆角，用于小型元素（标签、徽标） |
| radius/medium | 8px | 中圆角，用于按钮、输入框等标准组件 |
| radius/large | 12px | 大圆角，用于标准卡片、小型弹窗 |
| radius/xlarge | 16px | 超大圆角，用于大型卡片、页面弹窗、包含复杂组件的容器 |
| radius/full | 999px | 全圆角，用于圆形按钮、头像、药丸形状 |

> [!tip] CSS 变量引用
> 在代码中通过 `var(--radius/small)` 等方式引用，确保全局一致性。

---

## 嵌套圆角

当元素嵌套时，内层圆角应小于外层圆角，以保持视觉平衡。

> [!tip] 设计建议
> 内层圆角 = 外层圆角 - 内边距

例如：卡片使用 large 圆角，卡片内的按钮使用 medium 圆角。

---

## 使用原则

1. **一致性**：同类组件使用相同的圆角层级
2. **层级感**：外层容器圆角 ≥ 内层元素圆角
3. **避免混用**：同一视图中避免出现过多不同的圆角值
4. **平台适配**：iOS 倾向使用较大圆角，Android 可稍小
