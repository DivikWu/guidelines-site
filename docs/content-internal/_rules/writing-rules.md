# YDS Writing Rules

## Required Frontmatter
- title
- description
- section
- slug
- order
- status

## Link Rules
- Prefer absolute routes: /foundations/color
- Hash (#) only for same-page anchors

## Obsidian Syntax
- ❌ Disable ![[embed]]
- ✅ Allow [[wikilink]] (will be converted later)

## Assets
- Put images under content/assets
- Use relative paths: ./assets/xxx.png
